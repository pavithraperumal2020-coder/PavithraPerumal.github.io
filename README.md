# Personal Portfolio Website

A static personal portfolio website built with HTML5 and CSS3.

## Features
- **Responsive Design**: Mobile-first approach using Flexbox and Grid.
- **PWA Ready**: Includes `manifest.json` and offline fallback page.
- **Clean UI**: Modern Blue + Gray color theme.
- **No Dependencies**: Pure HTML & CSS (No JS, No Frameworks).

## Project Structure
```
portfolio-website/
│
├── index.html          (Home)
├── about.html          (About Me)
├── skills.html         (Skills)
├── projects.html       (Projects)
├── contact.html        (Contact)
│
├── css/
│   └── style.css
│
├── assets/
│   ├── images/         (Place your images here)
│   └── icons/          (Place PWA icons here)
│
├── manifest.json       (PWA manifest)
├── offline.html        (Offline fallback page)
└── README.md
```

## How to Use
1. **Icons**: Replace `icon-192x192.png` and `icon-512x512.png` in `assets/icons/` with your own.
2. **Images**: Add your project and profile images to `assets/images/`.
3. **Deploy**: Upload the `portfolio-website` folder to any static host (GitHub Pages, Netlify, Vercel).

## Customization
- Edit `css/style.css` to change colors (CSS Variables).
- Update HTML files with your personal content.
